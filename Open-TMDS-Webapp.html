TMDS Log Analyzer

This page opens a live TMDS webapp server.

The HTML file itself is not the parser or backend. For automatic updates, point this page at one central TMDS server that runs from your master copy.

  1. If this PC hosts the master tool, run TMDS-Server-Switch.bat here.
  2. If another PC hosts the master tool, enter its TMDS server URL below, for example http://YOUR-PC:4173/.
  3. Keep the master TMDS server running. Users who open this page will be sent to that live tool.
